# Sky Rogue: Aether Drift

A custom browser-based 3D sky sandbox built with `Three.js` and `Vite`.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- `WASD`: strafe and surge through the air
- `Mouse`: steer the ship
- `Space`: climb
- `C`: dive
- `Shift`: overdrive boost
- `Left Click`: fire
- `F` or `Right Click`: pulse blast

## Goal

Roam freely across the floating islands, harvest crystals, destroy sentry drones, and thread drift rings to clear the mission grid. After that, keep free-roaming and chase score.
